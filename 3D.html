🌟 Le Musée des Créations du Futur

Les dessins des enfants transformés en 3D

Chaque enfant a dessiné sur papier. Voici leurs créations devenues de vrais modèles 3D !

Pour ajouter une nouvelle création :
1. Place le fichier .glb dans le même dossier.
2. Ajoute un nouvel objet dans le tableau creations (voir les exemples ci-dessous dans le code).
3. Champs : enfant (obligatoire), titre (optionnel), description et glb.
